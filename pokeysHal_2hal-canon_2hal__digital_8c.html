Detailed Description

Canonical HAL export functions for digital input and output devices.

This file provides helper functions to export HAL pins and parameters conforming to the LinuxCNC Canonical Device Interface specification.

Supported interfaces:

  • Digital Input (digin)
  • Digital Output (digout)

All exported pins follow the canonical naming convention: <device>.<device-num>.<io-type>.<specific-name>

See: https://linuxcnc.org/docs/html/hal/canonical-devices.html
