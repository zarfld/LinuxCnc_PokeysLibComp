Configure a UART interface.

Issues command PK_CMD_UART_COMMUNICATION with subcommand 0x10. Parameter interfaceID selects the UART port while format contains the serial frame format bits. The desired baud rate value is placed in the request payload.

Parameters
deviceTarget device handle.
baudrateSerial baud rate in bits per second.
formatFrame format and options.
interfaceIDUART interface index.
Returns
Result code from SendRequest().
