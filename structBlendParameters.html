Detailed Description

9D Input geometry for a spherical blend arc. Blend arc parameters (abstracted). This structure holds blend arc parameters that have been abstracted from the physical geometry. This data is used to find the maximum radius given the constraints on the blend. By abstracting the parameters from the geometry, the same calculations can be used with any input geometry (lines, arcs, 6 or 9 dimensional lines).
