enum StateField

Enum for various fields of state info that are int type.

WARNING:

1) Since these are used as array indices, they have to start at 0, be monotonic, and the MAX_FIELDS enum MUST be last in the list.

2) If your application needs to pass state tags through NML, then you MUST update the corresponding cms->update function for state tags.

enum StateFlag

Enum to define bit names for StateTag's flags register. The actual position of the flag isn't important, so the actual index doesn't matter. However, the bit field should be <=64 bits to fit within a long int.
