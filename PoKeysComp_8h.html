Detailed Description

Main header file for the PoKeys HAL component in LinuxCNC.

This file defines the central data structure sPoKeysDevice, which represents a PoKeys device instance. It also declares all related functions used for IO access, PulseEngine v2 configuration, encoder reading, PoExtBus/PoNet communication, and component initialization.
