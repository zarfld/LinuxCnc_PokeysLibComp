◆ PK_EasySensorsSetupGet()

int32_t PK_EasySensorsSetupGet ( sPoKeysDevice *  device)

Retrieve EasySensors configuration from the device.

Each EasySensor present on the device is queried and the information is stored in the sPoKeysDevice structure. Sensor type, reading ID, refresh period, failsafe configuration and the unique sensor identifier are updated for every entry in the EasySensors array.

Parameters
devicePointer to an initialized device structure.
Returns
PK_OK on success or a negative ::PK_ERR code on failure.
