Detailed Description

SPI communication helpers (command 0xE5).

Provides basic configuration and data transfer routines for the SPI interface on PoKeys devices.

uint8_t  prescaler,
uint8_t  frameFormat 
)

Configure the SPI interface of a PoKeys device (command 0xE5/0x01).

Issues PK_CMD_SPI_COMMUNICATION with subcommand 0x01 to set the SPI clock prescaler and frame format.

Parameters
devicePointer to an initialised device structure.
prescalerSPI clock prescaler value sent to the device.
frameFormatFrame configuration (mode, bit length etc.).
Returns
PK_OK on success or a negative ::PK_ERR code on failure.

Receive data from the SPI interface (command 0xE5/0x20).

Issues a read request for the specified number of bytes. The returned values are copied into the provided buffer on success.

Parameters
devicePointer to an initialised PoKeys device structure.
bufferDestination buffer for the received bytes.
iDataLengthNumber of bytes to read (maximum 55).
Returns
PK_OK if the data was read successfully, otherwise a negative ::PK_ERR code.

Transmit data over the SPI interface (command 0xE5/0x10).

Sends a block of up to 55 bytes to the selected chip on the PoKeys device. The chip select line is asserted automatically by the device.

Parameters
devicePointer to an initialised PoKeys device structure.
bufferData buffer containing bytes to transmit.
iDataLengthNumber of bytes to send (max 55).
pinCSChip select pin identifier on the PoKeys device.
Returns
PK_OK on success or a negative ::PK_ERR code when the transfer fails.
