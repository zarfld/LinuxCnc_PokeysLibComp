Parameters
devicePointer to PoKeys device
cmdPoKeys command ID (e.g., 0xC4, 0xC5, 0xCE, etc.)
paramsOptional pointer to up to 4 parameter bytes (NULL if none)
params_lenLength of params (max 4 bytes)
payloadOptional pointer to payload (e.g., encoder options) to copy starting at byte 8
payload_sizeSize of payload (must fit into remaining packet space)
parser_funcOptional parser function for response (NULL for write-only operations)
Returns
Request ID on success, negative error code on failure
