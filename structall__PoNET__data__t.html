Detailed Description

Contains all PoNET-related data for communication with PoKeys devices.

This structure manages the state and HAL representation of all connected PoNET devices, including:

  • Up to 16 generic PoNET modules
  • A kbd48CNC keyboard module with 48 keys (each with LED and Button)
  • Status and control values like PWM and brightness
