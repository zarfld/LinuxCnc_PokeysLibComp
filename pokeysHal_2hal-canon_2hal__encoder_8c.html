Detailed Description

HAL export and function registration helpers for canonical encoder devices.

This file provides functions to export HAL pins and parameters and register a periodic read function for encoder channels that conform to the LinuxCNC Canonical Device Interface (CDI).

The encoder interface exposes both raw and scaled data to HAL, including position and velocity information, with index-reset logic support.

Functions provided:

  • hal_export_encoder() – defines HAL pins and parameters for an encoder
  • hal_register_encoder_func() – registers a periodic HAL function to read the encoder

All exported symbols follow the canonical naming format: <prefix>.encoder.<index>.<pin|param|func>

See also
https://linuxcnc.org/docs/html/hal/canonical-devices.html#sec:hal-cdi:encoder
