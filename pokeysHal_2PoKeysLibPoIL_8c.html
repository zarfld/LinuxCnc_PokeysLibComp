void PK_ParsePoILStateResponse ( sPoKeysDevice *  device)

Decode interpreter state response into the device structure.

The function expects that device->response already holds data returned by the PK_CMD_POIL_OPERATIONS command. It parses the contents and updates the sPoKeysDevice::PoIL fields accordingly.

Parameters
devicePointer to an initialized device structure that received the response.
