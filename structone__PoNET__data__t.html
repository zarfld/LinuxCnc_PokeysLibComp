Detailed Description

Structure holding HAL parameters and pin references for one PoNET device.

This structure represents a single PoNET module connected to the PoKeys device, including identification, addressing, and data exchange over the PoNET bus.

It supports:

  • Read-only identification and configuration parameters
  • 16 status input pins (HAL_OUT)
  • 16 status output pins (HAL_IN)
