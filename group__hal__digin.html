Canonical HAL interface for binary hardware inputs.

The canonical digital input (digin) provides a standardized interface for reading binary values from hardware. Drivers implementing this interface must provide the following pins:

  • in (BIT): state of the hardware input
  • in-not (BIT): inverse of the input state

No HAL parameters are defined.

The optional read function should update both pins from hardware input.
