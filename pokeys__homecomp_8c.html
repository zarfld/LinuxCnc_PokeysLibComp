Detailed Description

HAL component for managing homing sequence and state logic using PoKeys PulseEngine v2.

This component implements a state machine to control synchronized homing of multiple joints based on PoKeys Pulse Engine v2 capabilities. It interacts with other components by exchanging command and status signals and monitors joint-specific flags such as is_homed.

Features:

  • Homing coordination across multiple axes
  • HAL pin exports for command and status per joint
  • Integration with the PoKeys PulseEngine hardware abstraction
  • Debug and tracing via HAL pins or internal flags

The component is designed for use with LinuxCNC and PoKeys57E devices, and provides real-time feedback and control of the homing process using HAL-compatible pins.
