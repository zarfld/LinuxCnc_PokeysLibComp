Read the light sensor value previously requested.

Executes PONET_OP_GET_LIGHT_SENSOR with subcommand 0x30 to obtain the measured light value which is stored in sPoNETmodule::lightValue.

◆ PK_PoNETGetModuleStatus()

int32_t PK_PoNETGetModuleStatus ( sPoKeysDevice *  device)

Obtain previously requested module status bytes.

After a call to PK_PoNETGetModuleStatusRequest(), this function reads the 16 status bytes from the device using PONET_OP_GET_MODULE_DATA with subcommand 0x30. The bytes are copied to sPoNETmodule::statusIn.

Parameters
devicePointer to an initialized PoKeys device structure.
Returns
PK_OK on success, PK_ERR_NOT_CONNECTED if device is NULL, PK_ERR_TRANSFER on communication failure or PK_ERR_GENERIC when the response format is invalid.
